Installation - top

Follow the steps below to get started with your Site Template:

  • Open the servosell-main-files/HTML Folder to find all the Templates Files
  • You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  • Make sure you upload the required files/folders listed below:
    • HTML/css - Extra Stylesheets Folder
    • HTML/images- Images Folder
    • HTML/js - Javacripts Folder
    • HTML/css/main.css - Main Stylesheet File you can also rename it to style.css (for wordpress)
    • HTML/index.html - Index File/Homepage
    The other files can be used according to your preferences.
  • Now in order to compile SASS files into CSS files
    • First go to sass-lang.com
    • Then go to install page by clicking on install link on top of the page
    • Now scroll down, on your left side there are some applications use for compiling SASS files into CSS files
    • We recommend to download and install Koala application by clicking on Koala
    • After installed open Koala application click on "+" sign button on top left side of that application in order to add SASS folder of this project
    • You get two files on your application "main.scss" and "dashboard.scss"
    • Now right-click on main.scss select "Set Output Path" then go to css folder and double click on main.css and replace it and "Set Output Path" for "dashboard.scss" to "dashboard.css"
    • Now click on one of them you wish to compile
    • Once you click a file a slide open on right side of that app
    • Make sure "Auto Compile" and "Source Map" are checked
    • Then click on compile button on bottom of that slide, and minimize that app
  • You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

How To Use CSS / JS Files - top

CSS and JS both are important part of the website. we've included all the plugins css/js inside dashboard.css, and main.js files with proper commenting to avoid more http request to server and to make website faster. main.css Inside HTML > CSS you can find dashboard.css which contains all the css.

	<link rel="stylesheet" href="css/main.css">

Bootstrap.css Inside HTML > CSS you can find bootstrap.css which is the main part of the website as Servosell based on bootstrap 4.3 it is recommended to include bootstrap.css file right before main.css file.

	<link rel="stylesheet" href="css/bootstrap.css">

main.js Inside HTML > JS you can find main.js which contains all the custom and plugins code used in website.

	<script src="js/main.js"></script>

Now simply change the Codes according to your requirements

Changing Fonts - top

Change your Fonts on the Fly as we have included fonts in main.scss and dashboard.scss file. Servosell uses 2 Fonts namely: Poppins and Open+Sans from the Google Fonts Library. You can find the Linking to the Font Files in the scss/base/_base.scss file

	@import url(https://fonts.googleapis.com/css?family=Nunito:400,600,700
|Open+Sans:400,700&display=swap);

In order to change the Fonts, you will need to Edit the Above Links with your Custom Font if you plan to use a Google Font or Remove it complete if you plan to use a Self Hosted font. Here is an Example for using Self Hosted Fonts.

Google Maps - top

You can add Embedded Google Maps to any Page using the following setup:

Step 1:

Add the Google Maps Specific Scripts in the <main.js> Tag below the js/main.js script linking:

	<script src="https://maps.google.com/maps/api/js?key=YOUR-API-KEY"></script>

Step 2:

Add a div which will contain your Google Map:

	<div id="sl-locationmap" class="sl-locationmap"></div>
